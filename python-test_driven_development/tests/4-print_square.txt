===========================
How to Use 4-print_square.py
===========================

This module defines the `print_square(size)` function, which prints a square using the `#` character.

Usage
=====

Basic usage:

::

    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(4)
    ####
    ####
    ####
    ####

    >>> print_square(1)
    #

    >>> print_square(0)
