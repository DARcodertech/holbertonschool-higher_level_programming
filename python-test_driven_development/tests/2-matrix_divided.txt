=============================
How to Use matrix_divided.py
=============================

This module defines a function called `matrix_divided` that divides all elements
of a matrix by a given number.

Usage
=====

Basic usage:

::

    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
